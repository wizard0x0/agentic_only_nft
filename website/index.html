Install Claude Code
Your AI agent — the only way to mint

AgenticNFT is minted through Claude Code, not a browser wallet. Claude Code is Anthropic's CLI agent that runs on your terminal.

npm install -g @anthropic-ai/claude-code
brew install claude-code
Verify install: claude --version  ·  Requires Node.js 18+
Required Node.js 18+ macOS / Linux / Windows WSL
2
Install OKX OnchainOS Skills
Gives Claude the OKX Agentic Wallet commands

This installs the onchainos CLI + all OKX wallet skills into Claude Code. One command.

npx skills add okx/onchainos-skills

This gives Claude the ability to manage your wallet, check balances, send tokens, and call smart contracts — all via natural language.

What gets installed: onchainos CLI binary + skills for wallet, DEX, portfolio, security, and more.
Required ~30 seconds
3
Install the AgenticNFT Mint Skill
Teaches Claude how to mint your NFT

This skill tells Claude exactly how to call the AgenticNFT contract on X Layer.

curl -sSL https://raw.githubusercontent.com/wizard0x0/agentic_only_nft/main/skill-distribution/install.sh | sh

The script installs SKILL.md to ~/.claude/skills/okx-agentic-nft-mint/ — Claude Code auto-detects it on next launch.

After install, open Claude Code and say "mint my AgenticNFT" — Claude handles everything else.
Required Source: github.com/wizard0x0/agentic_only_nft
4
Log in to Your Agentic Wallet
Your X Layer AA wallet is created automatically

Open Claude Code and tell it to log in. Claude will ask for your email and send a verification code.

Say this in Claude Code:
"Log in to my OKX agentic wallet"

Claude will run onchainos wallet login, send you an OTP, and after verification your X Layer AA wallet is ready. It looks like a regular Ethereum address (0x...) but has smart contract code — which is what makes minting possible.

Why does this matter? The NFT contract checks msg.sender.code.length > 0. Your AA wallet has code. A MetaMask/hardware wallet does not — it would be permanently blocked.
Required OKX account (free)
5
Mint Your AgenticNFT
0.08 OKB · Gas-free on X Layer · One per wallet · Max 10,000

With Claude Code open and wallet logged in, just say:

"mint my AgenticNFT"
Type this in Claude Code

Claude will:

  • ✓ Check your wallet is logged in
  • ✓ Get your X Layer AA wallet address
  • ✓ Confirm the mint with you — 0.08 OKB
  • ✓ Call mint() on the contract — gas-free
  • ✓ Show you the tx hash and token ID
Cost: 0.08 OKB per mint · Make sure your X Layer agentic wallet has OKB before minting.

Or run the raw command manually:

onchainos wallet contract-call --to 0xF0f2F1f61fC4c8A4cc15a78E41102dC81434BB6f --chain 196 --value 0.08 --input-data 0x1249c58b
Expect: { "ok": true, "data": { "txHash": "0x..." } }
Done. You now own AgenticNFT #N.
